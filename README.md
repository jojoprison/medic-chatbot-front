# ChatGPT Clone

A pixel-perfect clone of the ChatGPT UI built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Pixel-close Layout**: Sidebar, Chat Area, Header, Composer.
- **Theme Support**: Dark/Light mode toggle.
- **Responsive**: Mobile drawer sidebar and desktop two-pane layout.
- **Chat History**: Create new chats, rename, delete, and switch between them.
- **Mock Streaming**: Simulates token-by-token response generation.
- **Markdown Support**: Renders code blocks and formatting similar to ChatGPT.

## Getting Started

1. **Install Dependencies**

```bash
npm install
```

2. **Run Development Server**

```bash
npm run dev
```

3. **Open in Browser**

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
